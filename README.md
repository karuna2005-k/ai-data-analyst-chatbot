# AI Data Analyst Chatbot 🤖

An intelligent chatbot built with LangChain and Groq that answers data analytics questions in simple, clear language.

## Features
- Conversational memory — remembers context across messages
- Data analyst personality — specialized in statistics, charts, and business insights
- Clear chat option to reset conversation
- Fast responses powered by Groq's LLaMA model

## Tech Stack
- Python
- LangChain
- Groq API (LLaMA 3.1)
- Streamlit

## How to Run
1. Clone this repository
2. Install dependencies:
pip install langchain langchain-groq langchain-core streamlit python-dotenv
3. Create a `.env` file and add your Groq API key:
GROQ_API_KEY=your-groq-api-key-here
streamlit run app.py
